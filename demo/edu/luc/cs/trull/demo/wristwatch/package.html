Demonstrates a simple wristwatch. The wristwatch uses the Trull stopwatch control logic unchanged as a subcomponent. This package also demonstrates event-driven unit testing within the JUnit framework. Furthermore, this package demonstrates the use of the Spring Framework for the following tasks:

  • building Trull components compositionally
  • hooking Trull components up with other components
  • accessing the resulting components via a factory object
